1. ECOMMERCE API

The E-Commerce API is a robust backend application built using JavaScript, Node.js, Express.js, and MongoDB. It facilitates the core functionalities required for an online shopping platform, including user authentication with JWT for secure login and role-based access. The API enables efficient product management, allowing the creation, retrieval, updating, and deletion of product data. It also supports order placement and tracking, ensuring seamless integration of customer and admin operations. With a focus on scalability and security, the API ensures smooth handling of data and transactions in a structured and efficient manner.
